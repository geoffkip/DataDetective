Team Titans: Data Detective

DataDetective aims to integrate all the economic, education, and health related data sets available on data.pa.gov. Our project is focused on visualizing multiple data sets at the same time to provide users with a way to explore the data.

  • View the data as a bar chart for different datasets.
  • The bar charts have double axes for viewing several measures.
  • Get recommendations from the recommendation engine to see other similar measures.
  • View geospatial data as a map.
  • Compare several different geospatial measures simultaneosly by viewing maps side by side./li>
  • Get recommendations to other similar measures using the reccomendation engine.
  • View time-series data as time-series graphs.
  • Compare several different time-series measures by plotting them on the same graph.
  • Filter the chart by using the year and month selector.
